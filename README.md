# countdown

A beautiful CLI countdown timer. Zero dependencies, pure Node.js.

## Install

```bash
npm install -g @alpha-studio/countdown
```

## Usage

```bash
countdown 60            # 1-minute countdown (MM:SS)
countdown 3600 -f hh:mm:ss  # 1 hour, full format
countdown               # interactive mode
```

## Options

- `-f, --format <fmt>` — Display format: `seconds` | `mm:ss` | `hh:mm:ss` (default: `mm:ss`)
- `-h, --help` — Show help
